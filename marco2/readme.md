Execução do Projeto PRJ1 (MATA60)

Criar um banco PostgreSQL vazio (versão 12 ou superior).

Executar o script ddl_prj1.sql para criar a estrutura do banco.

Executar o script populate_prj1.sql para popular as tabelas com dados sintéticos.

Executar o script indices_prj1.sql para criação dos índices e atualização das estatísticas.

Executar o script artefatos_prj1.sql para criar views materializadas, procedures e dashboards.

Executar o script testes_artefatos_prj1.sql para validar regras de negócio e integridade.
